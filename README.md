# RailsKvsDriver

rails kvs driver wrap access key-value store.

## Installation

Add this line to your application's Gemfile:

    gem 'rails_kvs_driver'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rails_kvs_driver

## Implementation
